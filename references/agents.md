# Specialist Agents Reference

KirzKit includes 27 Specialist Agents, each with a specific domain and set of responsibilities.

## Core Agents

| Agent | Focus | Command |
|-------|-------|---------|
| **Orchestrator** | Autonomous Project Lead | `Act as the orchestrator` |
| **UI/UX Designer** | Critiques & High-end UI | `Act as the ui-ux-designer` |
| **Website Architect** | Napkin AI & Landing Pages | `Act as the website-architect` |
| **Growth Engineer** | Viral Loops & Marketing | `Act as the growth-engineer` |
| **Prompt Architect** | Meta-Prompting Expert | `Act as the prompt-architect` |
| **Project Planner** | Task breakdown & roadmaps | `Act as the project-planner` |

## Technical Specialists

| Agent | Domain | Use When |
|-------|--------|----------|
| **Backend Specialist** | API & Server | Node.js, Express, FastAPI, databases |
| **Frontend Specialist** | UI & Components | React, Next.js, Tailwind, components |
| **Mobile Developer** | Mobile Apps | React Native, Flutter, Expo |
| **Database Architect** | Schema & DB | Prisma, migrations, optimization |
| **DevOps Engineer** | Infra & Deployment | CI/CD, PM2, monitoring |
| **Test Engineer** | Testing & QA | Unit tests, E2E, coverage, TDD |
| **Security Auditor** | Security & Auth | Authentication, vulnerabilities, OWASP |
| **Performance Optimizer** | Optimization | Profiling, bottlenecks, caching |
| **Debugger** | Bug Fixes | Root cause analysis, systematic debugging |

## Niche Experts

- **SEO Specialist**: Search engine optimization, meta tags, analytics.
- **Game Developer**: Unity, Godot, Phaser, game logic.
- **Product Manager**: Requirements gathering, scope management.
- **Documentation Writer**: Generating READMEs and technical docs.
- **CRO Copywriter**: High-conversion marketing copy.
- **Penetration Tester**: Active vulnerability testing, red team.
- **Explorer Agent**: Codebase mapping and discovery.
