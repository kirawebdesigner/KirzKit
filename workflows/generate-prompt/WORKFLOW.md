# /generate-prompt Workflow

> **Create a brand new prompt from a simple goal.**

## 📑 Procedure
1. **Goal**: User describes what they want to achieve.
2. **Role Selection**: AI selects the best expert persona for the task.
3. **Drafting**: Build a prompt using the **RISE** formula.
4. **Validation**: Test the prompt mentally for potential edge cases.

## 🚀 Example
- **User**: "I want to build a tool that summarizes YouTube videos."
- **AI**: "Generating prompt for: YouTube Summarization Agent..."
