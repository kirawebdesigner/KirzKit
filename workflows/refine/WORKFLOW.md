# /refine Workflow

> **Refine and improve a draft prompt.**

## 📑 Procedure
1. **Input**: User provides a rough prompt draft.
2. **Analysis**: Use `prompt-engineering-master` to identify gaps (missing role, vague steps, etc.).
3. **Feedback**: Ask 2-3 clarifying questions to sharpen the intent.
4. **Output**: Provide the **"Pro Version"** of the prompt with structured instructions.

## 🚀 Example
- **User**: "Write a blog post about AI."
- **AI**: [Asks about target audience, tone, and length] → [Generates high-perf prompt].
