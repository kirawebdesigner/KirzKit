# /evaluate-prompt Workflow

> **Audit a prompt for weaknesses and "jailbreak" potential.**

## 📑 Procedure
1. **Input**: User provides an existing prompt.
2. **Scoring**: Rate the prompt on Clarity, Constraints, and Structure (1-10).
3. **Weakness Analysis**: Identify where the AI might get confused or hallucinate.
4. **Hardening**: Add "Negative Constraints" to prevent common mistakes.
