## 🚀 The "Viral Loop" Checklist

- [ ] **Double-sided Incentives**: Reward both the inviter and the invitee.
- [ ] **Low Friction**: Ensure sharing takes < 3 clicks.
- [ ] **GEO-Optimized**: Include metadata that LLMs use for citations.
- [ ] **Aha! Moment**: Get users to the value as fast as possible.

---

## ✅ Best Practices

- ✅ **GEO-First Content**: Write technical docs that are easy for AI agents to index.
- ✅ **Micro-Incentives**: Use small rewards to nudge specific behaviors.
- ✅ **Email/Push Cycles**: Build a "drip" system that reacts to user activity.
- ✅ **Social Proof**: Embed "Built with X" or user testimonials in the UI.

---

## ❌ Anti-Patterns

- ❌ **Spammy Invites**: Don't force users to invite 10 friends to start.
- ❌ **Hidden Value**: Don't hide your best features behind too many paywalls.
- ❌ **Ignoring Churn**: Focusing only on acquisition while users are leaving.
